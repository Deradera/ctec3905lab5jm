The three languages of the web

HTML5

Defines the content of a web page.
Uses semantic elements to define what blocks of information are. (e.g. header/footer)
Classes and IDs to commonly group similar content so it can be styled the same.
No center tags in HTML5!
Simple DOCTYPE declaration.

A W3Schools intro to HTML5

CSS

  • Specifies the layout of a web page.
  • Capable of drawing shapes and doing simple animations
  • Has been split into modules (e.g. selectors/text effects/animations).
  • Completely backwards compatible with css.
  • Can use classes and ids to target specific content

A W3Schools intro to CSS3

JavaScript

Behaviour
Used to program the behaviour of a web page.
Changing HTML
Is capable of changing/hiding HTML elements and attributes (e.g. can change the img src upon clicking a button).
Changing CSS
Can change css attributes upon performing certain actions.
Show/Hide
Can be used to show and hide content upon hovering or clicking on tabs.
